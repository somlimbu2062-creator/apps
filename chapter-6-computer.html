Chapter 6: Web Technology - I

An introduction to the Internet and HTML.

Introduction to the Internet

The **Internet** is a global network of interconnected computers. It's a vast collection of networks that link devices worldwide, allowing for the exchange of data and information. The basic components required for an Internet connection include a **TCP/IP** enabled computer, an account with an **ISP**, a telephone line, a **MODEM**, and a computer with sufficient hardware capacity.

Internet Services and Applications

  • World Wide Web (WWW): A series of interconnected services through hypertext, providing links in web documents. It uses the **Hypertext Transfer Protocol (HTTP)**.
  • Web Browser: A software program that allows users to access and display web pages. Examples include Google Chrome, Mozilla Firefox, and Safari.
  • FTP (File Transfer Protocol): A tool for transferring files between computers over the Internet.
  • E-mail: The most widely used Internet feature for sending and receiving messages and digital data electronically.
  • Voicemail: A service that answers phone calls and records audio messages.
  • USENET: A bulletin board service (BBS) that organizes messages into thousands of newsgroups.
  • Chat: A feature for real-time text-based communication.
  • Video Conferencing: A service that enables real-time face-to-face communication with both sound and video.
  • URL (Universal Resource Locator): A unique address that identifies a specific location on the Internet, such as https://www.example.com.
  • E-Fax: A service to send and receive faxes directly on a computer via the Internet.
  • TELNET: Allows users to log in to a remote computer system.

Internet Terminologies

  • ISP: Internet Service Provider, a company that provides Internet access.
  • Link: A quick route to jump from one web page to another.
  • Search Engine: Software that searches websites for specific information (e.g., Yahoo, Google).
  • Domain Name and IP Address: A domain name (e.g., google.com) is a unique, human-readable name for a website, which corresponds to a numerical **IP Address**.
  • Web Server: A site where web pages are stored. It responds to requests from web browsers.
  • POP/SMTP Servers: **POP (Post Office Protocol)** servers store incoming mail, while **SMTP (Simple Mail Transfer Protocol)** servers transfer electronic mail between computers.
  • HTTP: A protocol that web browsers and web servers use to communicate.
  • Proxy Server: A server that improves access speed and updates information.
  • Internet Roaming: A facility to use your Internet account from a different country.

Example of an email address: krishna@wlink.com.np where krishna is the username, @ separates the username and domain, wlink is the ISP name, .com indicates a commercial organization, and .np is the country code.

Introduction to HTML

HTML stands for **HyperText Markup Language**. It's the language used to create web pages. An HTML file is a text file that contains several **tags** and **attributes** which are interpreted and displayed by a web browser. HTML was developed by **Tim Berners-Lee**.

Uses of HTML

  • To create the basic layout and structure of a web page.
  • To embed images, videos, and audio.
  • To format text (e.g., bold, italic, underline).
  • To create lists and hyperlinks.

Advantages of HTML

  • Easy to learn and use.
  • Supported by all web browsers and operating systems.
  • HTML files load quickly.

Disadvantages of HTML

  • Difficult to create attractive and dynamic web pages using only HTML.
  • Not a programming language.
  • Incompatibilities can exist across different versions.

HTML Tags and Attributes

An **HTML tag** is an instruction to the web browser to organize content. Tags are written inside angle brackets (<>). Tags and attributes are not case-sensitive.

  • Pair Tags: Tags that have both an opening tag (<tag>) and a closing tag (</tag>). Also known as **container tags**. Example: <body>...</body>.
  • Singular Tags: Tags that have only an opening tag and no closing tag. Also known as **empty tags**. Example: <br> (line break).

An **attribute** is a descriptive property of an HTML tag, written within the tag itself. Attributes add extra features to tags, allowing for customization. For example, `bgcolor` is an attribute of the `` tag to set the background color.

To create an HTML document, you write the code in a text editor like Notepad, save it with a `.html` extension (e.g., `sample.html`), and then open it with a web browser to execute.

Web Designing Terminologies

  • Webpage: An HTML document that displays information on the web.
  • Website: A collection of interrelated web pages, all connected through hyperlinks.
  • Homepage: The main, introductory page of a website.
  • URL: Uniform Resource Locator, the unique address of a web page.

DBMS (Database Management System)

Data are raw, unprocessed facts and figures. **Information** is meaningful data that has been processed. A **database** is a systematic collection of data. A **Database Management System (DBMS)** is a software system that manages the data stored in a database, allowing for quick and easy retrieval.

Advantages of DBMS

  • Quick and easy data searching.
  • Reduces data redundancy and inconsistency.
  • Enhances data security and privacy.
  • Allows data sharing over a network.

Disadvantages of DBMS

  • Higher installation and operating costs.
  • Requires more disk space.
  • Requires a technical professional (DBA) to maintain.

Define CSS. Explain its type.

CSS stands for **Cascading Style Sheets**. It is a style sheet language used to describe the presentation of a document written in a markup language like HTML. CSS defines how elements should be displayed on a web page, including colors, fonts, layout, and other visual aspects. It allows designers to separate the content (HTML) from its visual design (CSS), making websites easier to manage and update.

There are three main types of CSS:

1. Inline CSS

This type of CSS is applied directly to a single HTML element using the style attribute. It is useful for applying a unique style to a single instance of an element. For example: <p style="color: blue;">This text is blue.</p>

2. Internal CSS (or Embedded CSS)

This is defined within the <head> section of an HTML document using the <style> tag. It applies styles to the entire web page. This is a good method for single-page websites or when styles are unique to a specific page. For example:

<head>
  <style>
    body {
      background-color: lightblue;
    }
    h1 {
      color: navy;
    }
  </style>
</head>

3. External CSS

This is the most common and recommended method. Styles are defined in a separate `.css` file and linked to the HTML document using the <link> tag in the <head> section. This approach allows a single CSS file to control the styling of multiple web pages, which is highly efficient for large websites. For example:

<head>
  <link rel="stylesheet" href="styles.css">
</head>
