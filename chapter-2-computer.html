1's and 2's Complement

1's Complement

The **1's Complement** of a binary number is obtained by inverting each digit, changing every 1 to a 0 and every 0 to a 1.

Example: The 1's Complement of (1011)₂ is (0100)₂.

2's Complement

The **2's Complement** is calculated by adding 1 to the 1's Complement of a binary number.

Example: For the number (1010)₂,

1's Complement is (0101)₂.

2's Complement is (0101)₂ + (1)₂ = (0110)₂.

Subtraction using Complements

This method is crucial for performing subtraction in binary. The general rules for subtracting Y from X are as follows:

Using 1's Complement Using 2's Complement
1. Calculate the 1's complement of Y. 1. Calculate the 2's complement of Y.
2. Add the result from step 1 with X. 2. Add the result from step 1 with X.
3. If there is an extra bit (carry-out), remove it and add it to the remaining bits. 3. If there is an extra bit (carry-out), simply remove it.
4. If there is no extra bit, find the 1's complement of the result from step 2 and add a negative (-) sign. 4. If there is no extra bit, find the 2's complement of the result from step 2 and add a negative (-) sign.

Example 1: Subtract (1000)₂ from (111)₂

Given question is (111)₂ - (1000)₂.

A) Using 1's Complement:

1. Pad (111)₂ to have the same number of bits as (1000)₂, so it becomes (0111)₂.
2. 1's complement of (1000)₂ is (0111)₂.
3. Add (0111)₂ to (0111)₂:
    0111
  + 0111
  -------
   1110
4. There is no extra bit. So, we find the 1's complement of the result (1110)₂ which is (0001)₂.
5. Add a negative sign. The result is -(0001)₂.

B) Using 2's Complement:

1. Pad (111)₂ to have the same number of bits as (1000)₂, so it becomes (0111)₂.
2. 2's complement of (1000)₂ is (0111)₂ + (1)₂ = (1000)₂.
3. Add (1000)₂ to (0111)₂:
    1000
  + 0111
  -------
   1111
4. There is no extra bit. So, we find the 2's complement of the result (1111)₂ which is (0000)₂ + (1)₂ = (0001)₂.
5. Add a negative sign. The result is -(0001)₂.

Example 2: Subtract (11001)₂ from (11101)₂

Given question is (11101)₂ - (11001)₂.

A) Using 1's Complement:

1. 1's complement of (11001)₂ is (00110)₂.
2. Add (11101)₂ to (00110)₂:
    11101
  + 00110
  -------
  100011
3. There is an extra bit (the leading 1). Remove it and add it to the remaining bits:
    00011
  +    1
  -------
   00100
4. The result is (00100)₂.

B) Using 2's Complement:

1. 2's complement of (11001)₂ is (00110)₂ + (1)₂ = (00111)₂.
2. Add (11101)₂ to (00111)₂:
    11101
  + 00111
  -------
  100100
3. There is an extra bit (the leading 1). Simply remove it.
4. The result is (00100)₂.

Boolean Logic

Boolean Algebra is an algebra of logic that deals with binary variables and logical operations. It was introduced by **George Boole**. In this algebra, variables can have only one of two possible values: 0 (False) or 1 (True). This is the fundamental language of all modern digital computers.

  • Boolean Variable: A variable that can hold only one of the two values, 0 or 1.
  • Boolean Function: An expression with binary variables and operators that evaluates to a single binary result (0 or 1). Example: F = XYZ’ + XY.
  • Truth Table: A table that shows all possible input-output relationships for a logical operation or function.

Basic Boolean Operators

There are three basic operators in Boolean Algebra:

  • AND Operator (.): The result is 1 only if both inputs are 1. It's similar to multiplication.
  • OR Operator (+): The result is 0 only if both inputs are 0. It's similar to addition.
  • NOT Operator ('): This is an inverter. It gives the complement of the input (1 for 0, and 0 for 1).

Logic Gates

A **logic gate** is an electronic circuit that performs a specific logical operation. They are fundamental building blocks of digital computers.

Basic Logic Gates

  • AND Gate: Produces a high output (1) only when all inputs are high.
  • OR Gate: Produces a low output (0) only when all inputs are low.
  • NOT Gate: An inverter that takes a single input and produces its complement as output.

Derived Logic Gates

  • NAND Gate: A combination of NOT and AND gates. The output is the complement of an AND gate. It produces a low output (0) only when all inputs are high.
  • NOR Gate: A combination of NOT and OR gates. The output is the complement of an OR gate. It produces a high output (1) only when all inputs are low.
  • Exclusive OR (X-OR) Gate: Produces a high output (1) only when the inputs are different.
  • Exclusive NOR (X-NOR) Gate: Produces a high output (1) when the inputs are the same.
